video태그 - 쌍태그
속성 src : 잘 사용하지 않음, preload: 웹페이지를 열자마자 비디오 파일을 자동으로 로딩, poster="이미지소스" : 동영상에 대한 대체 이미지, controls: 영상제어버튼, autoplay : 웹페이지 열자마자 자동재생 mute,loop : 무한반복 재생
자식태그로는 source태그를 사용 source의 속성으로는 src="영상경로" type="video/파일형식 mp4, oggo webm" source태그는 fallback을 한다. 브라우저마다 지원하는 형식이 다르기 때문이다.
세가지 타입의 형식을 모두 사용해야 모든 브라우저에서 지원가능한다.
